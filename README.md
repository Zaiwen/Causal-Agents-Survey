# Causal-Agents-Survey
"This is a repository for storing relevant literature. We organize and categorize it into two main directions: **Causality for Agent** and **Agent for Causality**. We have also collected and supplemented literature on **Causality for Embedding**, **Embedding for Causality**, **Causality for Embodied AI**, and **Embodied AI for Causality**."
